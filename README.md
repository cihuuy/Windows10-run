# Windows10
WindowsSH is a Windows Emulator that runs on VNC.
The script to get these files is to type this commmand


` wget https://raw.githubusercontent.com/cihuuy/Windows10-run/main/win10start.sh `
` chmod +x win10start.sh `
` ./win10start.sh `

` chmod +x w10noq.sh `
` ./w10noq.sh `
